# GraphicViewer
## Описание
Деcктопное приложение для просмотра графических примитивов, считываемых из json файла
## Использование
Запустить приложение, выбрать файл с корректной схемой. Откроется окно с фигурами.
Схема в shapes.schema.json. Пример есть в test_examples/data.json.
## Сборка
Вариант 1:
 1. Запустить сборку VS 2 раза
 2. Установить расширение Conan в VS

Вариант 2:
 1. `conan install conanfile.py`
 2. Сборка в VS

Или создать файлы проекта для другой IDE через conan, или написать свой рецепт CMake.
